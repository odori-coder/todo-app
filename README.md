# ToDo アプリ

ブラウザだけで動く、シンプルな ToDo アプリです。ライブラリ不要・単一HTMLファイルで完結し、データは各ブラウザの localStorage に自動保存されます。

## 使い方

`index.html` をブラウザで開くだけです（ダブルクリック、または `file://` で開く）。

## 特長

- タスクの追加・完了・削除
- フィルター（すべて / 未完了 / 完了）
- 並び替え（追加順 / 締め切りが近い順 / 名前順）
- カテゴリー分け（色付きタグ表示・カテゴリーで絞り込み）
- 締め切りとカテゴリーを**文章に混ぜて入力**すると自動で反映
  - 例: `レポート提出 明日 #仕事` / `買い物 6/15 #プライベート` / `掃除 来週月曜`
  - 日付表現: `今日` `明日` `明後日` `来週` `来週月曜` `週末` `3日後` `2週間後` `6/15` `6月15日` `2026-06-15` など
  - カテゴリー: `#カテゴリー名`
- データはブラウザの localStorage に自動保存
